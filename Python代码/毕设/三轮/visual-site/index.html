数据来源与解释边界

本系统基于 CODEX 单细胞对象表和空间坐标构建，用于展示胃癌原发灶与卵巢转移灶的配对空间证据。

20μm 邻域指标用于表示局部空间接触机会或局部共定位关系，不等同于证明真实细胞相互作用，也不单独推断因果机制。

导览中的每个局部 ROI 是为了帮助理解论文证据链，正式结论仍以全体配对样本的定量指标、显著性分析和稳健性分析为准。
