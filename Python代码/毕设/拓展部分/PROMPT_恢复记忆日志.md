# PROMPT 恢复记忆日志

## 1. 项目背景

这是一个本科毕业设计项目，主题是：

**基于 CODEX 空间蛋白组学的原发灶 vs 转移灶配对可视化系统**

当前主线不是做通用病理平台，而是做一个适合本科答辩展示的交互式可视化 demo，并在此基础上补充一个“机器学习拓展分析”部分。

---

## 2. 当前主分析基线

已有 baseline 已基本完成，内容包括：

- 原发灶 vs 转移灶配对数据整理
- Tumor / M1 / M2 三类细胞提取
- 单细胞空间坐标标准化
- 空间指标计算
- R-only / L-only 分层比较

当前主结论是：

1. 转移灶整体更偏 M2 型构成
   - `diff_m2_frac_of_mac` 多数为正

2. Tumor-M1 互作下降
   - `diff_enrich_20/30_Tumor_M1` 多数为负或接近 0

3. M2 更外围化，而不是更贴肿瘤
   - `within_20/30_M2` 在多个 R-only 病例中为负

答辩 demo 当前也已经做过一版本地可运行的可视化系统。

---

## 3. 当前拓展分析方向

为了补充论文亮点，没有选择“识别原发灶和转移灶”的简单分类模型，因为这个方向意义较弱。

最终选择的拓展方向是：

**基于局部窗口的无监督聚类探索**

通俗理解：

- 不再只给每个样本算一个整体指标
- 而是把样本切成很多局部窗口
- 对每个窗口计算 Tumor / M1 / M2 的局部空间特征
- 再用聚类方法总结出几类典型局部微环境模式
- 最后比较这些模式在原发灶和转移灶中的占比差异

这个方向的意义是：

- 更适合补充主分析中的空间重排解释
- 更容易形成论文亮点
- 更适合和可视化系统结合

## 5. 当前拓展分析方法

### 5.1 窗口切分

- 基于归一化坐标
- 每个样本切成 `6 x 6` 局部窗口

### 5.2 窗口特征

当前为每个窗口计算：

- `tumor_density`
- `mac_density`
- `m1_to_tumor`
- `m2_to_tumor`
- `m2_mac_ratio`
- `m2_outer_bias`
- `m1_minus_m2`

其中：

- `m2_mac_ratio` 用来表示 M2 在巨噬细胞中的占比
- `m2_outer_bias = m2_mac_ratio - m2_to_tumor`
  用来粗略表达“虽然 M2 比例高，但不一定贴 Tumor，可能更外围化”

### 5.3 聚类方法

由于本机没有 `sklearn`，脚本里自己实现了轻量版：

- 标准化
- PCA
- KMeans

聚类数目前固定为：

- `N_CLUSTERS = 4`

## 6. 当前聚类结果

当前聚类中心在：

`tables/cluster_centroids.tsv`

目前得到的 4 类局部模式大致为：

1. `Tumor-dense immune-sparse`
   - 肿瘤很致密
   - 免疫细胞很少

2. `M2 outer-biased`
   - `m2_mac_ratio` 很高
   - `m2_outer_bias` 很高
   - M1 很弱
   - 更适合解释“转移灶中 M2 更偏外围化”

3. `Immune-rich M2-skewed`
   - 免疫细胞非常丰富
   - M2 偏高
   - 但更像免疫富集型，不一定贴主结论

4. `Mixed transition`
   - Tumor / M1 / M2 更混合

## 7. 当前可以得出的拓展结论

### 7.1 支持较强的部分

最强支持来自：

**`M2 outer-biased` 模式在 R-only 转移灶中占比升高**

根据当前输出：

- R-only 下，`M2 outer-biased` 的平均窗口占比变化为 `+0.151`

这说明：

- 转移灶中更容易出现“局部 M2 偏高且更外围化”的窗口模式

因此，这部分结果可以较好支持以下主结论：

1. 转移灶更偏 M2 型构成
2. M2 更外围化

### 7.2 支持较弱的部分

对于：

**Tumor-M1 互作下降**

当前聚类结果支持还不够强。

原因：

- 当前没有稳定分出一个特别明确的 `Tumor-M1 support` 类型
- 也没有形成“原发灶该模式高，转移灶该模式低”的强聚类链条

所以当前拓展分析对这条主结论：

- 只能算间接支持
- 不能替代主分析中的统计结果

## 10. 2026-04-08 基线结果复查与后续思路补记

### 10.1 当前对 baseline 三个结论的再判断

重新对 R-only 样本对做可视化后，发现：

1. **结论 1（转移灶更偏 M2 型构成）**
   - 总体方向仍然存在
   - 但不是所有 R 样本都一致
   - 其中 `6P vs 6R` 为反向，`13P vs 13R` 接近饱和

2. **结论 2（Tumor-M1 互作下降）**
   - 方向上仍有一定支持
   - 但受 `M1` 数量过小影响较大
   - 指标容易因为少量细胞波动而变化明显

3. **结论 3（M2 更外围化）**
   - 当前图形展示下支持力度不算强
   - 更适合表述为“存在总体趋势”
   - 不宜作为最硬的核心结论

### 10.2 样本 6 和样本 13 的具体情况

#### 样本 6

- `6P`: `M1 = 213`, `M2 = 688`, `M2/(M1+M2) = 0.76`
- `6R`: `M1 = 42`, `M2 = 37`, `M2/(M1+M2) = 0.47`

说明：

- `6R` 中巨噬细胞总数已经很少
- `M1` 与 `M2` 都非常少
- 此时比例指标很容易因少量计数变化而翻转
- 因此它更像“低计数高波动样本”，而不是稳定支持某个结论的样本

#### 样本 13

- `13P`: `M1 = 52`, `M2 = 3429`, `M2/(M1+M2) = 0.99`
- `13R`: `M1 = 32`, `M2 = 3223`, `M2/(M1+M2) = 0.99`

说明：

- 两边都几乎完全由 `M2-like` 主导
- 指标已经接近上限
- 因此该样本对对“转移灶更偏 M2”这个结论的增量贡献很弱
- 更适合作为“高度 M2 偏向样本”理解，而不是方向性证据

### 10.3 如何向导师询问 M1 问题

建议不要直接说“我是不是做错了”，而是从“指标稳健性”和“分类口径”去问。

可直接使用的表述：

“我这两天复查了 baseline 里的 R-only 配对，发现 `Tumor-M1` 互作和 `M2/(M1+M2)` 这两个指标对 `M1` 数量非常敏感。像 6R、13R 这类样本里，`M1` 计数本身很低，导致轻微的计数变化就会带来比例和互作指标的明显波动。结合文献里对 TAM 异质性和 M1/M2 二分法局限的讨论，我想请教您：这类低 M1 样本在 CODEX/空间组学研究里是否属于正常、可研究但需要谨慎解释的情况？如果是的话，我是否应该把当前结果定位为探索性结论，并增加绝对计数、稳健性分析，或者扩展到更多细胞类型后再下结论？” 

这个问法的好处是：

- 你是在讨论“方法是否稳健”
- 不是在暴露自己不懂基本概念
- 也自然引出了“是否需要扩展研究范围”这个更高层的问题

### 10.4 当前原始数据还能怎么理解和分类

从当前已整理出的分析表看，核心字段包括：

- 空间坐标：`Center X`, `Center Y`
- 标记物：`PanCK`, `CD68`, `CD86`, `CD163`
- 当前标签：`Tumor`, `M1`, `M2`

因此，除了现有的三分类，还可以考虑以下几种分类思路：

1. **更保守的 marker 组合分类**
   - `Tumor`: `PanCK+`
   - `Macrophage-like`: `CD68+`
   - `M1-like`: `CD68+CD86+`
   - `M2-like`: `CD68+CD163+`
   - `双阳/混合态`: `CD68+CD86+CD163+`
   - `其他非本研究重点细胞`: 不强行解释

2. **放弃硬性 M1/M2 二分，改成 macrophage 状态谱**
   - `CD86-high`
   - `CD163-high`
   - `CD86/CD163 mixed`
   - `CD68+ but low-polarization`

3. **按空间位置重新分类**
   - 靠近 Tumor 的 macrophage
   - 远离 Tumor 的 macrophage
   - Tumor core / edge / stromal-like 区域

4. **按局部微环境模式分类**
   - Tumor-dense immune-sparse
   - immune-rich M2-skewed
   - M2 outer-biased
   - mixed transition

### 10.5 后续实验思路

当前较合理的下一步不是简单推翻 baseline，而是做一个“稳健性升级版”。

建议的实验路线：

1. **先做稳健性检查**
   - 对 `M1`、`M2` 绝对计数单独作图
   - 标出低 `M1` 样本
   - 测试去掉极低 `M1` 样本后，三个结论是否仍成立

2. **将 `M1/M2` 改写为 `M1-like/M2-like`**
   - 避免表述过硬
   - 与文献中“体内 TAM 为连续谱”的观点一致

3. **引入更多分类层次，而不只局限于三类细胞**
   - 至少增加 `Macrophage-like / mixed-state macrophage / other cells`
   - 如果原始数据还有更多 marker，可进一步扩展免疫和基质细胞类型

4. **将主结论从“单指标”改成“绝对数量 + 比例 + 空间位置”联合判断**
   - 例如同时看 `M1_n`
   - `M2_n`
   - `M2/(M1+M2)`
   - `Tumor-M1` 邻域互作
   - `M2` 到 `Tumor` 的距离分布
